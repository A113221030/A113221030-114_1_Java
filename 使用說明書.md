\# 系統使用說明書

\#\# 環境需求  
\- Java JDK 17 或以上版本  
\- 支援 Java 的 IDE (如 IntelliJ IDEA, Eclipse)

\#\# 核心功能操作指引

\#\#\# 1\. 初始化平台與內容  
\[cite\_start\]透過 \`Platform\` 類別建立媒體庫。您可以將不同的內容物件（如 \`Movie\` 或 \`Series\`）加入平台 \[cite: 17\]。

\#\#\# 2\. 使用者訂閱管理  
\- \[cite\_start\]建立 \`User\` 時需指定初始訂閱方案（例如 \`FreeTier\`）\[cite: 15, 62\]。  
\- \[cite\_start\]系統支援方案升級：當 User 持有 \`PremiumPlan\` 時，可解鎖 4K 畫質與移除廣告功能 \[cite: 33, 55\]。

\#\#\# 3\. 權限檢查機制  
\[cite\_start\]在執行播放前，系統會自動透過 \`Restrictable\` 介面進行以下檢查 \[cite: 12\]：  
\- \*\*年齡限制\*\*：使用者年齡必須符合影片的分級要求。  
\- \[cite\_start\]\*\*地區限制\*\*：使用者所在地區必須在許可範圍內 \[cite: 8\]。  
\- \*\*方案限制\*\*：豪華內容僅限豪華方案使用者觀看。

\#\#\# 4\. 推薦與搜尋  
\- \[cite\_start\]使用 \`getSimilarContent()\` 方法可獲取與當前影片標籤（Category）相同的推薦清單 \[cite: 12\]。  
\- \[cite\_start\]系統會根據觀看歷史記錄，動態計算推薦分數 \[cite: 12\]。

\#\# 程式碼執行範例  
\`\`\`java  
// 範例：建立平台並執行權限播放  
Platform platform \= new Platform();  
User user \= new User("U001", new FreeTier()); \[cite\_start\]// 免費會員 \[cite: 30\]

Movie movie \= new Movie("奧本海默", 15, "Taiwan", true); \[cite\_start\]// 豪華限制內容 \[cite: 8\]  
platform.addContent(movie);

// 執行播放請求  
try {  
    movie.play(user);   
} catch (Exception e) {  
    System.out.println(e.getMessage()); // 會輸出「權限不足」  
}
